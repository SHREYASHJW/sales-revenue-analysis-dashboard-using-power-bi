# Sales & Revenue Analysis Dashboard

## Project Overview

This project is an interactive Power BI dashboard created to analyze sales performance, revenue trends, and top-performing products.

## Features

* KPI Visualizations
* Revenue Trend Analysis
* Top Performing Products
* Interactive Slicers and Filters
* Business Insights Dashboard

## Tools Used

* Power BI
* Excel/CSV Dataset

## Dashboard Components

* Sales KPI
* Profit KPI
* Revenue Trend Line Chart
* Product Performance Bar Chart
* Region & Category Filters

## Purpose

The dashboard helps businesses analyze sales data and make better decisions using data visualization.
